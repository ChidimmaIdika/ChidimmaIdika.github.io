Some tools I work with:

Microsoft Excel

Excel is a versatile and widely-used spreadsheet software that I leverage for data analysis, manipulation, and visualization. I use it to clean and preprocess data, perform basic statistical analysis, and create informative charts and graphs.

Tableau

Tableau is my go-to tool for interactive data visualization and reporting. With Tableau, I can transform complex datasets into insightful dashboards and visualizations, making it easier for stakeholders to understand and interpret data.

Microsoft Power BI

Power BI is an essential tool in my toolkit for creating dynamic and interactive data reports and dashboards. It allows me to connect to various data sources, transform data, and generate compelling visuals to drive data-driven decision-making.

SQL

(MySQL, Postgre SQL, SQLite, Microsoft SQL Server)

SQL (Structured Query Language) is my primary tool for database management and querying. I use it to retrieve, manipulate, and analyze data stored in relational databases, ensuring data accuracy and consistency.

Python

Python is my preferred programming language for data analysis and machine learning. I use it to develop data pipelines, perform advanced analytics, and build machine learning models. Its extensive libraries and packages make it a powerful tool for a wide range of tasks.

Machine Learning

Machine Learning is at the core of my data science work. I employ various ML algorithms and techniques to uncover patterns, make predictions, and extract valuable insights from data. This enables data-driven decision-making and the development of predictive models for various applications.

Heart Disease Prediction

This project is a comprehensive exploration of machine learning techniques to predict the presence of heart disease in individuals. Leveraging a diverse set of machine learning algorithms, this predictive model aims to provide accurate and actionable insights for early detection and intervention in heart disease cases.

Bank Fraud Detection Analysis

In this project, I delved into a comprehensive dataset from a bank containing valuable transaction records and certain information such as transaction type, amounts, customer details, and time stamps. My analysis aimed to uncover patterns and insights within the data using SQL, particularly focusing on distinguishing between legitimate and fraudulent activities.

By leveraging data-driven insights, I contributed to their commitment to enhancing customer experiences and continuous service improvements.

Netflix Movies Analysis using Python

This project involved analyzing Netflix movie data using Python. I investigated whether the average duration of movies on Netflix has been decreasing over the years by creating visualizations, cleaning and subsetting data, and drawing insights from the dataset. By leveraging Python's data analysis and visualization capabilities, I explored trends in movie durations and provided insights into the evolving landscape of movies on the Netflix platform. This project showcased my data analysis skills and proficiency in Python for exploring real-world datasets and drawing meaningful conclusions.

Nigeria's Economic Data
